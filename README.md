# Association Blelma Dream
This repository holds the codebase for the [Blelma Dream](https://www.facebook.com/profile.php?id=100066988150540) association bridging rural tradition and sustainable development in Tunisia.

## Tech Stack
* **Frontend:** Next.js (App Router, RTL Arabic primary)
* **Database:** Turso (via Drizzle ORM)
* **Styling:** Vanilla CSS
